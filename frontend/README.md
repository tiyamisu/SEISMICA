# SEISMICA — Frontend

React 19 single-page application for the SEISMICA command-center dashboard.

## Quick Start

```bash
npm install
npm start        # Development server on http://localhost:3000
npm run build    # Production bundle → /build
npm test         # Run test suite
```

> **Requires the backend to be running on port 5000.**
> See the [root README](../README.md) for full setup instructions.

## Key Dependencies

| Package | Version | Purpose |
|---|---|---|
| react | ^19 | UI framework |
| react-leaflet | ^5 | Interactive map |
| leaflet | ^1.9 | Map engine |
| recharts | ^3 | Charts & histogram |
| react-scripts | 5.0.1 | Build toolchain (CRA) |
